What is a function?

A function is a set of instructions grouped together to perform a specific task.

A program in Java has at least function. This is the main function.

Basically, there are two types of functions in Java, those of libraries (which are imported and used, such as: println, sqrt, round, nextLine ...) and those that are created by the user(developer). The ones we will work on in this block are the ones created by the user.

Functions created by the developer

Its syntax is as follows:

private static void functionName(){
   ................
   ................
}
public static void main(String[] args){
   ...........
   ...........
  functionName();
   ...........
   ...........
}

Structure

To work with functions in Java we need:

The creation of the function (it contains the code to perform the task):

private static returnType functionName (type (1) argument (1), .., type (n) argument (n))
{
  // body of function
}

The call within the main or other function to use it:

functionName (argument (1), .... argument (n));

Advantages

  • They help to break down the problem into small blocks of code that are easier to understand, maintain, and debug.
  • They avoid code redundancy if it has to be repeated.
  • The workload can be divided into different functions.

Sample

import java.util.Scanner;
public class Example {
  private static int add(int a, int b) {// function declarator
    / * Start of function definition. * /
    int total;
    total = a + b;
    return total; // return statement of function
    / * End of function definition. * /
  }

  public static void main(String [] args) {
    int num1, num2, sum;
    Scanner in = new Scanner(System.in);
    System.out.println("Enters two number to add");
    num1 = in.nextInt ();
    num2 = in.nextInt ();
    sum = add(num1, num2); // function call
    System.out.println("Sum =" + sum);
  }
}
